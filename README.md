This project rocks and uses MIT-LICENSE.

Reins can be used with or without turbolinks.



add to gemfile:

  gem 'reins'


add to application.js:

  //= require reins


add to layouts/*.html.erb, right before </body>:

  <%= reins_script_tag %>
